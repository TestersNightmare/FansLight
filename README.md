This flashlight app features a clean design and comprehensive functionality, optimized for emergency lighting and everyday use. Below are its core features and highlights, delivering an exceptional user experience.

<img alt="Logo" src="app/icon.png" width="120" />

## Core Features

### **Ultra-Bright Display**
Provides powerful illumination for dark environments, guiding direction or locating items in real-time.

### **Customizable Strobe Mode**
Users can freely adjust flashing frequency, supporting everything from rapid flashes to intermittent pulses for sending emergency distress signals.

### **Built-in SOS Mode**
One-tap activation of SOS mode provides assistance in critical situations.

## Additional Highlights

### **Color-Switching Display**
Freely adjust light color based on environmental needs to avoid glare or suit different scenarios like parties or emergency alerts.

### **Quick Launch**
After launching the app, the device remains active, with LED lights automatically or manually activated.

### **Widget Support**
Offers a 1x1 widget with customizable color and transparency for quick flashlight access.

## User Experience

### **Clean Design**
Employs Material Design principles with a dark theme for an exceptional visual experience.

### **Privacy Protection**
The app supports offline operation, contains no ads or unnecessary permissions, and ensures data security.

### **Fully Open-Source Support**
Completely open-source with customization support, allowing users to adjust color schemes and more.
